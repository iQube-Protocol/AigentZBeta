# Stack Architecture Overview

## Layers
### metaMe
Personal policy, delegated authority, guardian, runtime, and economic boundary layer.

### AgentiQ OS
Machine-legible substrate for agent discovery, capability exposure, trust, pricing, receipts, and orchestration.

### AgentiQ
Organizational and vertical cartridge shaping engine.

### Fuel
- iQubes for context and composability
- Qc for pricing, metering, budgets, attribution, and rebates

### Integrity
- DVN receipts for proof and lineage
- remote, deferred, and canonical minting for value finalization

## Working cartridge pair
- KNYT Cartridge = primary live cartridge
- Qriptopian Cartridge = signal, editorial, and amplification support cartridge
