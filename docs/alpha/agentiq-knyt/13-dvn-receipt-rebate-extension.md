# DVN Receipt and Rebate Extension

## Definition
DVN receipts prove that meaningful events occurred.
The rebate extension determines whether those events are reward-eligible, attributable, disputable, and finalizable.

## Core principle
Receipt-driven rebates, not click-driven rebates.

## Must support
- receipt identity
- event lineage
- actor attribution
- policy and trust snapshots
- Qc linkage
- provisional vs finalized state
- dispute and reversal support
