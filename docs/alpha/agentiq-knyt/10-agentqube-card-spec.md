# AgentQube Card Spec

## Purpose
AgentQube Card is the machine-legible wrapper for each agent.

## It should carry
- identity
- interfaces
- capabilities
- trust posture
- policy posture
- economics posture
- context references
- receipt behavior
- cartridge overlays

## It should wrap
- A2A-style discovery
- MCP-linked capabilities
- ERC-8004 trust surfaces
- AgentiQ policy semantics
- Qc economics
- DVN receipt behavior

## Usage
Every production agent should have one canonical AgentQube Card.
