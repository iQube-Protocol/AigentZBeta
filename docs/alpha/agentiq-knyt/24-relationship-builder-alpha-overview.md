## What this is

**Relationship Builder α** is the new Venture Lab α tab for the **AVL Relationship Builder** workstream.

It packages the product, continuity, implementation, and operator materials for replacing the current Marketa Cartridge product surface with a broader relationship orchestration surface.

In this model:

- **AVL Relationship Builder** is the venture-studio relationship surface
- **Marketa** remains the lead relationship agent
- **KNYT Relationship Composer** becomes the first live program workspace inside AVL
- **KNYT** is treated as an ongoing program
- the **Kickstarter** is treated as **Campaign Wave 1** of that program

---

## Why this exists

The current Marketa implementation inside AigentZBeta is already the correct stack-native base, but it still behaves primarily like a unified admin console.

Relationship Builder α reframes that work into a more durable product model:

- two-sided internal and partner workflows
- Studio-powered relationship composition
- Experience Matrix and NBE-aware actions
- Make as one execution rail among several
- KNYT as the first live program
- continuity beyond Kickstarter close

This is not a parallel system and not a return to the old Lovable portal as production. It is the Next.js-native evolution of that workflow model.
