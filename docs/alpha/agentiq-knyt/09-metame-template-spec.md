# metaMe Template Spec

## Definition
metaMe Template is the personal sovereignty template that lets an individual shape authority, guardian behavior, context exposure, runtime behavior, skill access, and economic boundaries.

## Core controls
- guardian mode
- authority classes
- lead agent
- budget posture
- receipt visibility
- context and memory policy
- skill visibility

## Alpha default posture
- Know1-led
- soft confirm
- low spend autonomy
- curated skills only
- receipt visibility on
- explanation-first native asset exposure

## Concise line
metaMe Template is how the user teaches the system how to act for them without giving up control.
