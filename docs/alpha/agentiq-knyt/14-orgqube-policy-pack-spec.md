# OrgQube Policy Pack Spec

## Definition
OrgQube Policy Pack is the machine-readable institutional governance envelope for agents, skills, cartridges, authority classes, budgets, trust requirements, and receipts.

## Governs
- allowed agents
- allowed skills
- allowed cartridges
- authority classes
- trust thresholds
- budget and spend rules
- native asset exposure posture
- required receipts
- escalation and dispute behavior

## Purpose
It lets institutions teach AgentiQ how it is allowed to operate.
