# Aigent Z Reference Spec

## Definition
Aigent Z is the reference agent for machine-legible, policy-aware, trust-scored, receipted execution.

## Core jobs
- resolve capabilities
- route work
- enforce policy
- check trust and budget posture
- invoke tools and workflows
- emit and verify receipts

## Position
Z sits in AgentiQ OS but operates inside metaMe and AgentiQ policy envelopes.

## Concise line
Z finds, evaluates, routes, executes, accounts for, and proves work.
