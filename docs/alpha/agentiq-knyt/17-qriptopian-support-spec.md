# Qriptopian Support Cartridge Spec

## Definition
Qriptopian is the support cartridge that creates signal, context, and editorial framing around the ecosystem, then routes qualified attention into KNYT and the wider stack.

## Primary roles
- widen the field
- create signal
- explain context
- support soft onboarding
- amplify launches and themes
- route into KNYT

## Canonical line
Qriptopian prepares and amplifies.
KNYT activates and retains.
