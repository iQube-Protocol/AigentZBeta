# Relationship Builder α

## What this bundle is

This is the Relationship Builder α doc bundle for the Venture Lab α cartridge.

It captures the product, continuity, customer-surface, seeded cohort activation, and implementation workback for replacing the user-facing Marketa Cartridge concept with **AVL Relationship Builder**, while keeping **Marketa** as the lead relationship agent and making **KNYT Relationship Composer** the first live program workspace.

## Current product decisions

- **AVL Relationship Builder** replaces the old Marketa Cartridge in product framing.
- **Marketa** remains the lead relationship agent inside AVL.
- **KNYT Relationship Composer** is the first live program workspace.
- **KNYT** is an ongoing program.
- The live Kickstarter is **Campaign Wave 1** of that broader KNYT program.
- Internal AVL must support both **partner** and **customer** communications and progression.
- **KS Backers** is a seeded customer cohort within the broader KNYT KS campaign.
- Stage 1 is **email-first activation** of that seeded cohort.
- Stage 1.5 is **list hygiene, grooming, and canonization**.
- Stage 2 is **cohort enrichment and expansion** using organic, swaps, and later paid/social/SMS.
