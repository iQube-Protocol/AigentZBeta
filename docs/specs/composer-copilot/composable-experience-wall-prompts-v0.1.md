# Composable Experience Wall Prompts v0.1

## Purpose

Provide reusable visual prompts for image and video generation experiences that describe the evolving system as a wall of production-grade ExperienceQube blocks.

These prompts are intended to:

- communicate the product vision clearly
- give a vivid visual metaphor for the composable system
- work as seed prompts in image or video generation experiences

## Core Visual Metaphor

The system is a wall of luminous, production-grade experience blocks.

Each block is:

- complete
- trustworthy
- composable
- production-ready

Together they form a larger structure:

- an experience wall
- a modular creative system
- a living runtime architecture

The visual language should suggest:

- premium runtime design
- sovereign infrastructure
- elegant modularity
- cinematic composition
- stacked or interlocking experience units

## 1. Base Prompt

Create a cinematic visual of modular ExperienceQube blocks being assembled into an experience wall. Each block should feel like a finished, production-grade composable unit with premium surfaces, glowing interfaces, subtle receipt/provenance indicators, and sovereign runtime quality. The wall should suggest that image generation, video generation, article creation, and deployment can each exist as standalone blocks or interlock into richer experiences. The composition should feel elegant, futuristic, architectural, and runtime-grade rather than abstract or chaotic.

## 2. Image Prompt Variant

Create a premium editorial image showing a wall of luminous ExperienceQube blocks being assembled like advanced creative lego bricks. Each block should represent a complete production-grade experience component such as image generation, video generation, article generation, or deployment. The blocks should look refined, trustworthy, and composable, with subtle runtime UI surfaces, provenance cues, and receipt-like marks. The wall should feel cinematic, architectural, and modular, expressing the idea that sophisticated experiences are built by combining finished interoperable units.

### Portrait Addendum

Vertical composition, strong focal cluster of blocks, premium lighting, clear depth, elegant runtime polish, suitable for a mobile-first experience surface.

### Landscape Addendum

Wide cinematic composition, strong sense of an expanding modular wall, clear architectural depth, premium runtime polish, suitable for a desktop or tablet runtime surface.

## 3. Video Prompt Variant

Create a concise cinematic video of production-grade ExperienceQube blocks sliding, locking, and illuminating as they assemble into a larger experience wall. Each block should feel like a fully developed composable unit representing image generation, video generation, article creation, or deployment. The motion should emphasize precision, modularity, trust, and elegant runtime quality. The result should feel like a sovereign creative system forming in real time from finished interoperable building blocks.

### Motion Guidance

- keep the motion readable and deliberate
- emphasize block assembly and interlocking behavior
- avoid chaos or overly mechanical factory motion
- show the wall becoming more capable as new blocks connect
- maintain premium runtime atmosphere

## 4. Qriptopian Variant

Create a cinematic Qriptopian-themed visual of modular ExperienceQube blocks assembling into a luminous experience wall. The wall should blend premium runtime technology with editorial and narrative creativity. Some blocks should suggest image generation, video generation, article creation, codex experiences, and deployment pathways. The result should feel like a sovereign modular creative system for the Qriptopian universe, elegant, trustworthy, and ready to be composed into larger runtime experiences.

## 5. metaKnyts Variant

Create a cinematic metaKnyts-themed visual of production-grade ExperienceQube blocks assembling into a larger experience wall. The blocks should feel like premium composable units for story capsules, video moments, imagery, codex experiences, and deployment. The scene should express high-trust modular creativity within the broader Qriptopian context, with strong visual identity, runtime polish, and a sense that richer experiences can be built by connecting complete interoperable pieces.

## 6. System Framing Copy

Use this short framing line when needed in UI or prompting:

**Build the wall from finished bricks.**

Alternative:

**Compose larger experiences from production-grade ExperienceQube blocks.**

## 7. Prompting Guidance

When using these prompts:

- keep the metaphor concrete rather than abstract
- avoid generic “floating cubes in space” imagery
- favor interlocking modular architecture
- keep the mood premium, composed, and trustworthy
- make each block feel like a finished runtime-grade unit

The goal is not just to show cubes.
The goal is to show a system of completed composable experience blocks forming a larger creative structure.
