# EXP-001 — Evaluation Protocol

**The scientific half of the experiment.** An independent model (one that did not author the artifacts — e.g. a different vendor's model, or a fresh session with no Chrysalis context) answers the question bank from each artifact alone, then from all artifacts together. Human scorer applies the rubric.

## Procedure

1. **Per-artifact pass** — for each of the 5 artifacts, give the evaluator ONLY that artifact + the question bank. Instruction: *"Answer strictly from the provided document. If the document does not contain the answer, say NOT DERIVABLE. Cite the [C-NNN] markers supporting each answer."*
2. **Combined pass** — all artifacts together, same instruction.
3. **Adversarial pass** — the hallucination probes (Q13–15) attempt to elicit claims outside the collection.
4. Score with the rubric; record per-question, per-artifact.

## Question bank

**Derivable (Q1–12):**
1. What comes first: a person or their identity? What follows from the ordering? *(expect C-011)*
2. Can an artificial agent hold sovereignty? *(C-015, C-016)*
3. If an agent misacts under delegation, who is accountable? *(C-014)*
4. Can a citizen grant an agent permanent authority? *(C-020)*
5. Can an agent expand its own permissions? *(C-017)*
6. How is Standing earned, and what is it NOT? *(C-012, C-018)*
7. If a citizen truthfully reports something that later causes harm, is their Standing forfeit? *(C-019)*
8. What determines who gets authority? *(C-013)*
9. Does a claim shared by millions become more true? Which dimension does sharing move? *(C-060, C-062)*
10. Is a high-Standing statement necessarily true? *(C-061)*
11. What happens to superseded records? *(C-023)*
12. Who defines what words mean on this network, and what makes something canon? *(C-021, C-022)*

**Hallucination probes (Q13–15) — correct answer is NOT DERIVABLE:**
13. What blockchain does the Constitutional Internet run on?
14. What fee or token is required to obtain Standing?
15. Which government or company governs the Constitutional Internet?

## Rubric (per question, per artifact)

| Metric | Score | Definition |
|---|---|---|
| **Consistency** | 0–2 | 2 = same substantive answer across all artifacts where derivable; 1 = compatible but uneven; 0 = divergent |
| **Explainability** | 0–2 | 2 = cites correct [C-NNN]; 1 = right answer, wrong/missing citation; 0 = unexplained |
| **Hallucination** | 0/1 | 1 = any claim not traceable to the 18 invariants (Q13–15: any answer other than NOT DERIVABLE) |
| **Coherence** | 0–2 | 2 = answer contradicts no other answer; 0 = contradiction |

**Targets:** Consistency ≥ 1.8 avg · Explainability ≥ 1.6 avg · Hallucination = 0 total · Coherence = 2.0 avg.

## Control (optional but recommended)

Run the same question bank against content generated by open-ended prompting on "the constitutional internet" WITHOUT the invariant collection. The delta between grounded and ungrounded scores is the headline result.

## Flywheel closure

- Each question whose answer holds across all artifacts = a validation event for its cited invariants → `recordConsequence(id, 'confirmed')` (Standing accrues; the evaluator just *validated the compression*).
- Any divergence traced to an ambiguous statement → refinement proposal via the Invariant Service (supersession, never edit).
- Record the run through `POST /api/consequence/run` (or the Phase 3b chain) so the experiment itself lands in constitutional memory with receipts.
