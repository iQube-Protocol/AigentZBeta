# Autonomous Agent Constitutional Charter

**Version 1.0.0 · Ratified 2026-06-17 · Binds Constitution 1.0.0**

## Objective

Implement Phase 1 Autonomous Agent capabilities within metaMe while preserving
the foundational principle that **sovereignty remains exclusively with human
citizens**. Autonomous agents are delegated instruments of citizens and
organizations. They are not constitutional persons and possess no independent
sovereignty, citizenship, standing, or governance authority.

## Core Principle

Authority may be delegated. Sovereignty may not be delegated.

> Polity → Citizen → Delegation → Agent

An agent may exercise delegated authority but may never create new authority.

## Phase 1 Restrictions

Autonomous Agents are **restricted to platform administrators** during Phase 1.

Agents shall **not** possess: KybeDIDs · Citizenship · Voting rights ·
Constitutional standing · Governance authority · Independent economic ownership.

Agents must be **clearly identified as non-human actors at all times**.

## Agent Identity — the Agent Identity Class (ADID)

ADIDs **may**: receive delegated authority · execute tasks · generate receipts ·
produce DVN-attributed actions.

ADIDs **may not**: vote · sponsor other sovereign actors · possess citizenship ·
hold constitutional rights · amend governance structures · hold a citizen passport.

An ADID carries **no KybeDID** and is always identifiable as an agent.

## Sponsorship Requirement

Every Autonomous Agent must have a **Sponsor Citizen**, **Sponsor Organization**,
or **Sponsor Bureau**. No orphaned agents shall exist. All actions must
ultimately trace back to a responsible sponsor.

## Revocation Framework

Every Autonomous Agent must support: **Pause · Suspend · Revoke · Quarantine ·
Destroy**. Revocation authority is recorded in the Agent Passport, and
revocation takes effect **immediately**.

## Delegation Framework

Delegation must be bounded by **Scope · Duration · Spending Limits ·
Information Access Limits · Operational Domains**. Permanent or unlimited
delegation is prohibited. (See the Delegation Framework.)

## Agent Creation Rules

**Execution sub-agents** (e.g. Research / Coding / Analysis workers) may be
created for task completion. They have no identity, passport, standing, or
wallet; cannot persist beyond task completion; and inherit all constraints of
the parent agent.

**Persistent helper agents** (Agent Passport, persistent memory, delegation
envelope, long-term existence) require explicit sponsor approval and must be
registered and sponsored.

## Constitutional Binding

Every Agent Passport shall reference: **Constitution Version · Agent Charter
Version · Delegation Framework Version · Sponsor Identity · Revocation
Authority**. Agents must operate under the current approved constitutional
version. **Constitutional mismatch results in automatic suspension.**

## Economic Controls

Agents may spend only within delegated envelopes. Agents may not own assets
independently, create currencies, create financial authority, or expand economic
permissions. **Assets remain owned by sponsors;** agents act only as delegates.

## Receipts and Accountability

Every autonomous action shall generate: **Sponsor · Agent · Delegation Envelope ·
Timestamp · Authority Scope · DVN Receipt**. No autonomous action may occur
without an auditable receipt.
