**Status:** canonical

Five alpha-active matrix instances mapped to the dual-axis model.

## Matrix 1 — Outsider → Acolyte

**Family:** Awareness + Patronage / Satisfaction + Endorsement  
**Patronage movement:** Outside the Order → Acolyte  
**PCS movement:** Observer → Collector  
**Macro intent:** World entry, first affiliation, first patronage

### Typical entry states
curious, unaware, skeptical, disinterested, mildly critical, aesthetically intrigued but not committed

### Target state
intrigued, oriented, first sense of belonging, willing to take a first concrete step into the Order

### Journey moment
First discovery, first landing in Runtime or Qriptopian, first exposure to KNYT lore/content/offer

### Active levers

| Tier | Levers |
|------|--------|
| Structural | KNYT intro content, storyworld framing, first collectible/entry artifact, Qriptopian article/landing, KNYT teaser/onboarding flow |
| Emotional | Intrigue, mystery, belonging, purpose, narrative coherence, trust via clarity and provenance |
| Transactional | Low-friction entry point, Acolyte path, first patronage prompt, starter reward or symbolic unlock |
| Governance | Public access, lightweight persona or wallet prompt if needed, no high-bar permissions |

### Signals to capture
dwell/read/watch, clickthrough, opt-in, first wallet/persona step, first purchase/patronage step, return visit, initial like/spark

### Signal meaning
Intrigue, initial trust, first belonging impulse, willingness to convert from observer to participant

### System response
clarify next step, simplify entry into Acolyte, surface "what happens if you join the Order", invite first participation

### Reward effect
symbolic first-entry recognition, lightweight badge/status/visibility, optional starter $KNYT reward

### Permission effect
unlock Acolyte-only pathways, unlock first participation surfaces

### Next-best steps
join as Acolyte → explore KNYT lore → view order path → perform first interaction

### Recovery steps (skeptical/critical users)
show provenance, clarify what KNYT is, reduce hype language, emphasize contained world and transparent progression

## Matrix 2 — Acolyte → Keta Participant

**Family:** Awareness + Patronage / Loyalty + Rewarded  
**Patronage movement:** Acolyte → Keta KNYT  
**PCS movement:** Collector → Curator  
**Macro intent:** First identity consolidation and first participation habit

### Typical entry states
excited but uncertain, newly enrolled, low confidence, exploratory, interested but not yet habitual

### Target state
identity-aligned early member, repeat participant, basic world fluency, willing to vote/like/spark and return

### Journey moment
immediately after entry, first few sessions as Acolyte, first attempts to understand value and belonging

### Active levers

| Tier | Levers |
|------|--------|
| Structural | Onboarding sequence, participation prompts, simple social/community actions, order progression explainer, early collectible/context surfaces |
| Emotional | Belonging, recognition, clear orientation, low-risk participation confidence, early loyalty formation |
| Transactional | Keta progression path, starter rewards, visible progression feedback, participation-linked unlocks |
| Governance | Acolyte permissions, Keta-eligible participation surfaces, low-risk prompts only |

### Signals to capture
first vote, first like, first spark, 2nd/3rd return session, first clear preference signal, content completion, low-friction opt-ins

### Signal meaning
early habit formation, willingness to do more than passively consume, comfort with world interaction, readiness for loyalty elevation

### System response
reinforce belonging, show progress toward Keta, surface "people like you also…", invite lightweight curation behavior

### Reward effect
first real $KNYT reward moment, visible progress tracker, recognition for repeat participation

### Permission effect
unlock deeper Keta experiences, unlock basic curation surfaces, reveal more world-specific pathways

### Next-best steps
complete first participation streak → curate preferred content/item → deepen patronage → progress into Keta

### Recovery steps (disinterested after entry)
reduce complexity, show immediate utility/value, reinforce why the Order matters, offer one simple high-value interaction

## Matrix 3 — Keta → Keji Curator/Remixer

**Family:** Awareness + Patronage / Engagement + Contribution + PCS  
**Patronage movement:** Keta KNYT → Keji KNYT  
**PCS movement:** Curator → Remixer  
**Macro intent:** Shift from participation to active shaping behavior

### Typical entry states
engaged, identity-forming, comfortable with world rules, interested in expressing preference or taste, beginning to want more agency

### Target state
active curator, early remixer, contributor-adjacent participant, more deeply invested patron

### Journey moment
after repeat participation, once basic belonging is established, once user shows preference depth and return behavior

### Active levers

| Tier | Levers |
|------|--------|
| Structural | Curation surfaces, remix prompts, world feedback mechanisms, community participation threads, editorial/canonical response opportunities |
| Emotional | Expression, taste, recognition, identity deepening, "my presence matters here" |
| Transactional | Rewards for meaningful participation depth, higher-tier patronage incentives, remix-linked recognition |
| Governance | Keta-to-Keji eligibility, permissioned remix surfaces, role-gated participation depth |

### Signals to capture
repeated voting with specificity, curation choices, saving/organizing preferences, remix attempts, commentary depth, community interaction quality

### Signal meaning
growing agency, developing taste, willingness to shape rather than merely consume, readiness for Keji elevation

### System response
present curation tools more prominently, unlock remix prompts, connect user to correspondents/community pathways, show how deeper engagement changes their role

### Reward effect
more meaningful $KNYT rewards for curated/remixed activity, status or recognition tied to quality of engagement

### Permission effect
unlock richer participation surfaces, unlock contributor-adjacent prompts, unlock Keji progression opportunities

### Next-best steps
curate a set → remix a world element → participate in themed community action → deepen patronage into Keji

### Recovery steps (stuck at passive engagement)
reduce friction to remix, present guided remix rather than open remix, use Kn0w1 to contextualize why curation matters

## Matrix 4 — First KNYT → Contributor / Worldbuilder

**Family:** Awareness + Patronage / Engagement + Contribution + PCS  
**Patronage movement:** First KNYT → Zero KNYT candidate  
**PCS movement:** Creator → Correspondent / Worldbuilder  
**Macro intent:** Contribution and visible world-shaping participation

### Typical entry states
highly aligned, identity-rich, already participating repeatedly, creatively activated, seeking greater purpose/status/influence

### Target state
real contributor, worldbuilder/correspondent, recognized participant in KNYT's evolving world, elite patron moving toward stewardship

### Journey moment
after clear repeat engagement, after remix/creation signals, once a user shows creative seriousness or contribution readiness

### Active levers

| Tier | Levers |
|------|--------|
| Structural | Creator prompts, submission pathways, correspondent/community contribution surfaces, community/worldbuilding initiatives, advanced codex/world access |
| Emotional | Meaning, co-authorship, prestige, stewardship identity, belonging through contribution |
| Transactional | Meaningful contribution rewards, higher-order recognition, access privileges, progression toward elite patronage |
| Governance | Contribution permissions, review/acceptance rules, editorial/canonical boundaries, trust and readiness gating |

### Signals to capture
original creation, structured submissions, correspondent behavior, high-quality remix, leadership in community, sustained contribution quality

### Signal meaning
transition from participant to contributor, readiness for deeper privilege, increased sovereignty and responsibility

### System response
elevate contribution tools, clarify standards, show contributor pathway explicitly, connect contribution to world impact and order progression

### Reward effect
higher-value $KNYT rewards, visible reputation/status markers, unlock contributor-recognition surfaces

### Permission effect
unlock advanced contributor capabilities, unlock candidate path toward Zero-tier stewardship

### Next-best steps
submit world-relevant contribution → become active correspondent → participate in advanced community initiative → move toward Zero KNYT / stewardship profile

### Recovery steps (high intent but low readiness)
provide guided templates, provide mentor/correspondent-style prompts, show examples of accepted contribution patterns

## Matrix 5 — Zero / Satoshi → Steward / Franchise-Aligned Sovereign Contributor

**Family:** Loyalty + Rewarded / Stewardship + Ecosystem Contribution  
**Patronage movement:** Zero KNYT → Satoshi KNYT  
**PCS movement:** Steward → Franchise-aligned Sovereign Contributor  
**Macro intent:** Apex stewardship, world leadership, franchise alignment

### Typical entry states
elite patron, highly trusted, highly aligned with world ethos, already contributing meaningfully, seeking deeper sovereignty and strategic role

### Target state
recognized steward, ecosystem-level contributor, franchise-aligned actor, world-shaping participant at the apex layer

### Journey moment
after sustained elite patronage, after repeated contribution, after visible leadership/stewardship behaviors, when franchise-level alignment readiness emerges

### Active levers

| Tier | Levers |
|------|--------|
| Structural | High-trust access surfaces, strategic world/treasury/franchise dialogue, stewardship tools, direct ecosystem contribution pathways, upstream contribution bridges |
| Emotional | Legacy, guardianship, sovereignty, leadership, deep purpose alignment |
| Transactional | Elite recognition, strategic reward logic, franchise-aligned economics, long-horizon incentive alignment |
| Governance | Highest trust thresholds, strict permissions, stewardship obligations, role-specific disclosure/access conditions |

### Signals to capture
sustained contribution over time, patronage depth, leadership behaviors, stewardship actions, strategic alignment, franchise readiness

### Signal meaning
the user is no longer just a participant or contributor — they are a world-level or ecosystem-level steward candidate

### System response
surface steward responsibilities, clarify apex pathways, connect KNYT role to broader ecosystem contribution, align with future 21 Sats / franchise responsibilities

### Reward effect
apex recognition, long-horizon alignment benefits, stewardship-linked economic signaling

### Permission effect
unlock stewardship/franchise-aligned pathways, unlock direct ecosystem shaping surfaces

### Next-best steps
formal stewardship path → franchise exploration → strategic ecosystem contribution → upstream collaboration with broader AgentiQ layers

### Recovery steps (high-patronage but low stewardship readiness)
distinguish patronage from stewardship, require visible responsibility and contribution markers, keep apex states earned not purchased alone
