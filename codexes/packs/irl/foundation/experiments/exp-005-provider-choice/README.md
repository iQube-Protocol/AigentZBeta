# EXP-005 — Provider Choice (outcome stability across providers)

**Family:** Foundational Validation Series (extension) · **Status:** run complete; canonical publication via the Results record

## Aim
Test **outcome stability across interchangeable model providers at equal grounding** — that a grounded task yields substantively the same result regardless of which provider serves it. Where EXP-004 asks *"does it hold on a sovereign provider?"*, EXP-005 asks *"is the outcome invariant to provider choice?"* — the property that makes provider selection an operational, not epistemic, decision.

## Method
The same grounded task runs across ≥2 interchangeable providers at identical grounding; per-arm provider/model is recorded and never merged. The experiment scores agreement of the grounded outcomes across arms — high agreement supports the claim that the constitutional substrate, not the model, carries the result.

## Honest limits
A model is still a model: agreement is measured, not guaranteed, and the provider set is bounded to those tested. Broader multi-provider coverage is a scale-up step.
